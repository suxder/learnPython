# ReviewNote

## 一、Python基础

### （一）数据类型

#### 1.数字（Number）

- 整数（int）
- 浮点数（float）
- 布尔值（bool）
- 复数（complex）

#### 2.组合数据（Composite Date）

- 字符串（string）

- 列表（list）
- 元组（tuple）
- 字典（dictionary）
- 集合（set）

> 数据类型归根结底是现实数据类型的映射，一个好的编程语言的数据类型应该简洁而有力。

#### 3.数据类型的可变与不可变性

- 不可变数据的变量内容改变后，内存地址改变。
- 可变数据的变量内容改变后，内存地址不变。

| **数据类型** | **可变不可变** |
| :----------: | :------------: |
|     数字     |     不可变     |
|    字符串    |     不可变     |
|     列表     |      可变      |
|     元组     |     不可变     |
|     字典     |      可变      |
|     集合     |      可变      |
